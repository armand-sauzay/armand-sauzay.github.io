- üìù Create and edit markdown notes
- üóÇÔ∏è Organize notes in folders
- üé® Beautiful TUI with syntax highlighting
- üì± Responsive layout with adjustable sidebar
- üóëÔ∏è Archive unused notes
- ‚å®Ô∏è Vim-style keybindings
- üéØ Focus mode without sidebar
- üîç Preview markdown rendering
- ‚ö° Fast and lightweight


### Keybindings

- `j/k` or `‚Üë/‚Üì`: Navigate notes
- `h/l` or `‚Üê/‚Üí`: Collapse/expand folders
- `enter`: Edit note/rename folder
- `n`: Create new note

- `backspace`: Archive note/folder
- `q` or `ctrl+c`: Quit
